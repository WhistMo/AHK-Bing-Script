
How to setup:

1. Install ahk.
2. Put search-term.txt to your C directory.
3. Modify the script as your like. If you have 5 accounts, you need to install 5 web browers. The script use IE, Chrome, Firefox, Opera and Safari. Make sure the path is the same in your computer.
4. Login in your bing account in each web browers, select Remember me in the login in page.

Above steps only need to do one time.

How to use:

Double click the Bing_Search.ahk
